    1. What is Semantic HTML? 
	Semantic HTML is HTML that has meaning to it. You name things that make sense.
    2. What is HTML used for?
	HTML is used to diplay text on websites. 
    3. What is an attribute and where do we put it? 
	An attribute is something we want to change. We use it in css between the {}
    4. What is the h1 tag used for? How many times should I use it on a page?
	The h1 tag is used to dictate what the most important thing is. We should only use it once per page.
    5. Name two tags that have required attributes
	The <a> tag and the <img> tag.
    6. What do we put in the head of our HTML document? 
	The title, style, base, link, meta, and script if you have them.
    7. What is an id? 
	An ID is a tag that we use when we want to style one thing specifically.
    8. What elements can I add an id to? 
	You can add an ID to whatever element you want.
    9. How many times can I use the same id on a page? 
	You can use the same ID only once per page.
    10. What is a class? 
	A class is similar to an ID, except its when you want to style different elements to the same and can make he changes all at once.
    11. What elements can I add a class to? 
	Just like the ID you can add it to pretty much any element.
    12. How many times can I use the same class on a page? 
	You can use a class as many times as you want per page.
    13. How do I get my link to open in a new tab?
	You would add target="_blank" in the a tag. 
    14. What is the alt attribute in the image tag used for? 
	That is the text that displays if the image cannot.
    15. How do I reference an id?
	You would reference an id with a # before it.
    16. What is the difference between a section and a div
	They are pretty much the same. We would use section more than div because it makes more sense.
    17. What is CSS used for? 
	CSS is used to style webpages.
    18. How to we select an element? Example - every h2 on the page
	You would select it like this: h2{}
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
	A class is comething when you want multiple things styled the same. An ID is when you want one thing styled the same. Say you have three paragraphs and you have a class for the first and last paragraph and an ID for the second paragraph.
    20. How do we select classes in CSS?
	You select classes in css with a .
    21. How do we select a p element with a single class of “human””?
	.human p{}
    22. What is a parent child selector? When would this be useful? 
	It selects every element that is a child of that element. It would be useful if you wanted to select children that are there and not all elements that match.
    23. How do you select all links within a div with the class of sidebar?
	.sidebar div{}
    24. What is a pseudo selector?
	It specifies a specific state of the selected element. For example :hover
